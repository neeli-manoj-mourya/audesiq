Accessibility

Contrast Pairs

#FFFFFF on Primary#5B4DFF bg✓ AA
#1E1E1E on CC Overlayrgba(255,255,255,0.9)✓ AAA
#1E1E1E on Accent#FFD400 bg✓ AA
👆
Touch Targets
Min 44×44 dp for all controls. Play button 64 dp. Sync button 52 dp height. Badges min 28–32 dp height.
📝
Subtitle Scaling
Default 24sp. Allow user to scale to 28–32sp in app settings. Use at least 3–4 visible lines in CC container.
🔊
Screen Reader Labels
Provide aria-label / contentDescription for: Play, Pause, Sync (with %, "Syncing 42%"), AD toggle, CC toggle, Back.
🎨
Color + Shape
Never use color alone for state. AD/CC badges use both color fill AND text label. Seek progress uses filled track.
🔄
Reduced Motion
Respect prefers-reduced-motion. Disable logo pulse, loader spin animations for users with vestibular disorders.
🧭
Focus Order
Ensure semantic focus order: Back → Title → AD/CC → Seek → Skip Back → Play → Skip Forward → Sync.
